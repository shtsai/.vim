# .vim
This repo contains my Vim configuration.

## Installation

Use the following commands to apply the configurations.
```
cd ~/
git clone https://github.com/shtsai7/.vim.git
ln -s ~/.vim/vimrc ~/.vimrc
cd ~/.vim
git submodule init
git submodule update
```
